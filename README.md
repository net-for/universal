# Universal RP CEF System

## рЃљрЃдрЃгрЃћрЃарЃљ
рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃарЃБрЃџрЃўрЃљрЃЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃБрЃарЃў CEF (Chromium Embedded Framework) рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ Universal RP рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА. рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ, рЃџрЃљрЃЏрЃљрЃќрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Pawn рЃЎрЃЮрЃЊрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ.

## рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

### ­Ъј« HUD рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ**: рЃАрЃљрЃ«рЃћрЃџрЃў, рЃџрЃћрЃЋрЃћрЃџрЃў, рЃцрЃБрЃџрЃў, рЃЉрЃљрЃюрЃЎрЃў
- **рЃ»рЃљрЃюрЃЏрЃарЃЌрЃћрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃ»рЃљрЃЋрЃерЃљрЃюрЃў**: рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЉрЃљрЃарЃћрЃЉрЃў
- **рЃќрЃЮрЃюрЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃў**: рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ/рЃАрЃљрЃерЃўрЃерЃў рЃќрЃЮрЃюрЃљ
- **рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў рЃЊрЃарЃЮ**: рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЊрЃарЃЮ
- **рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў**: рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў

### ­ЪЊ▒ рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў**: рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЊрЃўрЃќрЃљрЃўрЃюрЃў
- **рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў**: рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃў, рЃЉрЃљрЃюрЃЎрЃў, рЃбрЃљрЃЦрЃАрЃў, GPS, рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў
- **рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃюрЃЮрЃЏрЃарЃћрЃЉрЃў**: рЃќрЃљрЃарЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃЉрЃљрЃбрЃљрЃарЃћрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў**: рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃў

### ­ЪЈд рЃЉрЃљрЃюрЃЎрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃцрЃБрЃџрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ**: рЃюрЃљрЃдрЃЊрЃў рЃцрЃБрЃџрЃў рЃЊрЃљ рЃЉрЃљрЃюрЃЎрЃўрЃА рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў
- **рЃАрЃљрЃ«рЃџрЃўрЃА рЃцрЃБрЃџрЃў**: рЃАрЃљрЃ«рЃџрЃўрЃА рЃцрЃБрЃџрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
- **рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃЉрЃљрЃџрЃљрЃюрЃАрЃў**: рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃЉрЃљрЃџрЃљрЃюрЃАрЃўрЃА рЃЕрЃљрЃарЃўрЃфрЃ«рЃЋрЃљ

### ­ЪџЌ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃўрЃА HUD
- **рЃФрЃарЃљрЃЋрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў**: рЃЕрЃљрЃарЃЌрЃБрЃџрЃў/рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў
- **рЃцрЃљрЃарЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў**: рЃЕрЃљрЃарЃЌрЃБрЃџрЃў/рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў
- **рЃАрЃљрЃгрЃЋрЃљрЃЋрЃўрЃА рЃЊрЃЮрЃюрЃћ**: рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЉрЃљрЃарЃў
- **рЃЏрЃљрЃюрЃЦрЃљрЃюрЃўрЃА рЃ»рЃљрЃюрЃЏрЃарЃЌрЃћрЃџрЃЮрЃЉрЃљ**: рЃ»рЃљрЃюрЃЏрЃарЃЌрЃћрЃџрЃЮрЃЉрЃўрЃА рЃЉрЃљрЃарЃў

### ­ЪЊЮ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ/рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- **рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў, рЃърЃљрЃарЃЮрЃџрЃў, рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃљ
- **рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ**: рЃърЃљрЃарЃЮрЃџрЃўрЃЌ рЃерЃћрЃАрЃЋрЃџрЃљ
- **рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ**: рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

### ­Ъњ╝ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃцрЃарЃћрЃўрЃЏрЃћрЃЉрЃў**: рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- **рЃбрЃљрЃЦрЃАрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**: рЃбрЃљрЃЦрЃАрЃўрЃА рЃЏрЃЮрЃгрЃЮрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ
- **GPS рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**: рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ

### ­Ъј» рЃЦрЃЋрЃћрЃАрЃб рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- **рЃЦрЃЋрЃћрЃАрЃб рЃцрЃарЃћрЃўрЃЏрЃћрЃЉрЃў**: рЃЦрЃЋрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
- **рЃърЃарЃЮрЃњрЃарЃћрЃАрЃў**: рЃЦрЃЋрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃарЃћрЃАрЃў

### 2. Pawn рЃЎрЃЮрЃЊрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ
рЃЎрЃЮрЃЊрЃў рЃБрЃЎрЃЋрЃћ рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ `universal.pwn` рЃцрЃљрЃўрЃџрЃерЃў:

- HUD рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

### 3. CEF Plugin
рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ рЃарЃЮрЃЏ рЃњрЃљрЃЦрЃЋрЃЌ CEF plugin рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ.

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

### CEF Events
```pawn
// рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
cef_emit_event(playerid, "Hud:pinfo", CEFINT(health), CEFINT(armor));

// рЃцрЃБрЃџрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
cef_emit_event(playerid, "game:CEF:money", CEFINT(money));

// рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ
cef_emit_event(playerid, "data:not", CEFSTR("рЃАрЃљрЃЌрЃљрЃБрЃарЃў"), CEFSTR("рЃбрЃћрЃЦрЃАрЃбрЃў"), CEFSTR("success"), CEFINT(1), CEFINT(3000));

// рЃќрЃЮрЃюрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
cef_emit_event(playerid, "Hud:zona", CEFINT(1)); // 1 = рЃАрЃљрЃерЃўрЃерЃў, 0 = рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ
```

### JavaScript Events
```javascript
// CEF event-рЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
window.addEventListener('message', (event) => {
    const data = event.data;
    if (data.type) {
        universalRP.handleCEFEvent(data.type, data);
    }
});

// Pawn-рЃќрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ
window.cef.emit('eventName', 'data');
```

## рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### HTML/CSS/JS рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ
- `index.html`: HTML рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
- `styles.css`: CSS рЃАрЃбрЃўрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў
- `script.js`: JavaScript рЃџрЃЮрЃњрЃўрЃЎрЃљ

### Pawn рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ

## рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ

### рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў CEF Events
- `data:pool:reg` - рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ/рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- `Hud:zona` - рЃќрЃЮрЃюрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў
- `data:not` - рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў
- `data:vehicle` - рЃЏрЃљрЃюрЃЦрЃљрЃюрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- `bank:info` - рЃЉрЃљрЃюрЃЎрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
- `job:frame` - рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃцрЃарЃћрЃўрЃЏрЃў
- `quest:frame` - рЃЦрЃЋрЃћрЃАрЃб рЃцрЃарЃћрЃўрЃЏрЃў

### рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў

- `F2` - рЃЉрЃљрЃюрЃЎрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ
- `ESC` - рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ

## рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў

### рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
